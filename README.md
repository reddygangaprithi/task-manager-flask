# 🚀 Task Manager - Full-Stack Flask Web App

A modern, responsive task management application built with **Python Flask**, **SQLite database**, and **Bootstrap CSS**.


## 📱 Demo Instructions
1. `pip install -r requirements.txt`
2. `python app.py`
3. Visit `http://localhost:5000`
4. Register → Login → Add tasks!

## ✨ **Live Features**
- ✅ **User Authentication** - Register/Login/Logout
- ✅ **CRUD Operations** - Create/Update/Delete personal tasks
- ✅ **Priority System** - High/Medium/Low with color badges
- ✅ **Due Dates** - Date picker + formatting
- ✅ **Responsive Design** - Mobile + Desktop
- ✅ **Modern UI** - Glassmorphism cards + gradients
